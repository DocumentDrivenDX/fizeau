## Description
FEAT-021 defines the multi-node dashboard UI. The existing React SPA uses flat routes (/beads, /documents, /graph). This epic replaces that routing structure with the node/project-aware URL scheme, adds the combined cross-project views, adds the project picker, and wires the commit log view to the new commits endpoint.

## Acceptance Criteria
The web UI implements bookmarkable routes /nodes/:nodeId, /nodes/:nodeId/beads, /nodes/:nodeId/sessions, /nodes/:nodeId/projects/:projectId/{beads,documents,graph,commits,sessions}; a project picker updates the URL on project change; combined bead and session views merge data from all registered projects client-side; the root / redirects to /nodes/:nodeId using GET /api/node
