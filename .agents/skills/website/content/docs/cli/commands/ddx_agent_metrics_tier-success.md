## ddx agent metrics tier-success

Report success rate per harness/model tier from execution evidence

### Synopsis

Scan .ddx/executions/*/result.json and aggregate execution outcomes
into per-tier success rates. A tier is identified by harness alone when the
result has no model field, or by harness/model when a concrete model is
recorded. Success is defined as outcome == "task_succeeded".

```
ddx agent metrics tier-success [flags]
```
